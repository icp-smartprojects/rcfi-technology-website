- **Background:** Primary Blue (#0A3D62)

- **Primary Button:**
  - Background: Primary Blue 600 (#082f4e)
  - Text: White
  - Hover: Primary Blue 700 (#062440)

  - Background: Secondary Teal (#1ABC9C)
  - Text: White

  - Background: Error Red 600 (#c93c2f)
  - Text: White

- **Border (default):** Neutral 400 (#d4dce5)

## ðŸ”„ State Transitions

### Button States
```
Default:    #0A3D62 (Primary)
Hover:      #062440 (Primary 700)
Active:     #051a2d (Primary 800)
Disabled:   #d4dce5 (Neutral 400) + opacity 0.5
Focus:      Ring of Primary 200 (#b3cde5)
```

### Link States
```
Default:    #1ABC9C (Secondary)
Hover:      #126b65 (Secondary 700)
Visited:    #169481 (Secondary 600)
Active:     #0d4239 (Secondary 800)
Focus:      Underline + Ring
```

### Input States
```
Default:    Border Neutral 400 (#d4dce5)
Focus:      Border Primary 600 (#082f4e) + Ring
Error:      Border Error 500 (#E74C3C) + Error 50 background
Success:    Border Success 500 (#2ECC71) + Success 50 background
Disabled:   Border Neutral 300 (#e8ecf1) + opacity 0.6
```
